# Projeto em Java puro

O projeto tem como proposta um bot em console que ajuda o usuário a iniciar seus estudos em programação. 

## ✨ Conceitos aprendido
- Lógica de programação
- Tipos de dados
- Loops
- Array
- Classe e Herança
